> Codex 与 Claude 获得更强的推理档位，远程连接、Work 启动和运行时恢复也更加稳健。

## ✨ New

- Codex 与 Claude Agent 现在支持 Ultra 推理档位，可在受支持的模型上启用运行时最高级别的推理与编排能力。

## 💎 Improvements

- Work 创建后会由 Server 直接根据提交的目标启动首次运行，减少客户端交接造成的延迟与不一致。
- 基于 Relay 的远程主机会在连接中断后采用带抖动的指数退避自动重连，并在 Cradle Server 重启后恢复已配置的连接。
- Pull Request 详情页改用紧凑且固定的标签栏，可更快地在摘要、时间线、检查与文件之间切换。

## 🐞 Fixes

- 修复创建远程工作区会话时未保留所选 Provider、模型与推理档位的问题。
- 修复 Codex 阻塞目标恢复路径与系统指令传递，使自动恢复和自定义指令能够可靠生效。
