> 经过约两个月的开发，Cradle Desktop 的第一个公开开发版本正式开放测试。

Cradle 是一个 AI 原生的桌面客户端，专注于为 Agent 驱动的工作流提供稳定、实用的运行环境。我们的目标不是发明一套全新的工作方式，而是自然地融入大家日常已经在使用的工作流。

这个版本标志着更大范围测试的开始。应用还没有经过大规模验证，bug、边界情况和粗糙之处都在预期之内，我们非常欢迎反馈。

## 我们想验证什么

如果 Codex 的某些体验让你觉得特别顺手，或者 Claude Code 在某些方面做得更好，我们都想听听。Cradle 还很早期，许多产品决策仍然是开放的。

## 当前状态

Cradle 目前的核心是可靠性和易用性，还没有特别花哨的功能。当前的支持以 Codex 工作流为中心；Claude Code 的支持已经可用，但部分工作流可能仍会遇到兼容问题。

一些已知的限制：

1. 应用体积目前还偏大
2. 打包和分发仍在优化
3. Apple Developer ID 签名尚未就绪
4. 自动更新机制仍在调研

感谢每一位愿意花时间测试一个未完成产品的人。每一个 bug 报告、每一次工作流讨论，都会实实在在地影响 Cradle 接下来的方向。
